## About This Project
This project incorporates interactive designs and a modern blue color theme, creating an engaging user experience.

### Features
- **Responsive Design:** Adjusts beautifully on all screen sizes.
- **Interactive Elements:** Hover effects and animations for an immersive experience.
- **Modern Styling:** Using CSS transitions and transformations for animations.
- **Improved Visual Hierarchy:** Clear headings and sections for better readability.

## Getting Started
To get a local copy up and running follow these simple steps.

### Prerequisites
- npm
- Node.js

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/JLNerecina/JLNerecina.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Usage
Run the application using:
```sh
npm start
```
